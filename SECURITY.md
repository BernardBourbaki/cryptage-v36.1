# Politique de Sécurité

## Versions Supportées

| Version | Support          |
| ------- | ---------------- |
|  37.0   | ✅ Supportée     |
| < 37.0  | ❌ Non supportée |

## Signaler une Vulnérabilité

Si vous découvrez une vulnérabilité, créez une **Issue** sur GitHub avec le tag [SECURITY].

Merci de ne pas divulguer publiquement avant qu'un correctif soit disponible.

## Algorithmes

- **AES-256-GCM** : Standard NIST
- **Argon2id** : Vainqueur Password Hashing Competition 2015
- **OpenSSL** : Bibliothèque cryptographique éprouvée
